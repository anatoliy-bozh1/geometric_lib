# Geometric Lib

## Общее описание

**Geometric Lib** — это учебная библиотека, реализующая функции для вычисления площади и периметра основных геометрических фигур: круга, квадрата, прямоугольника и треугольника.  
Библиотека разработана в рамках лабораторных работ по дисциплине *«Инструментальные средства разработки программного обеспечения»* и предназначена для демонстрации принципов структурирования и документирования кода.

---

## Описание функций

### Файл `rectangle.py`

- **area(a, b)** — вычисляет площадь прямоугольника.  
  **Пример:** `area(3, 4) → 12`
- **perimeter(a, b)** — вычисляет периметр прямоугольника.  
  **Пример:** `perimeter(3, 4) → 14`

---

### Файл `triangle.py`

- **area(a, h)** — вычисляет площадь треугольника по основанию и высоте.  
  **Пример:** `area(6, 4) → 12`
- **perimeter(a, b, c)** — вычисляет периметр треугольника.  
  **Пример:** `perimeter(3, 4, 5) → 12`

---

### Файл `square.py`

- **area(a)** — вычисляет площадь квадрата.  
  **Пример:** `area(5) → 25`
- **perimeter(a)** — вычисляет периметр квадрата.  
  **Пример:** `perimeter(5) → 20`

---

### Файл `circle.py`

- **area(r)** — вычисляет площадь круга.  
  **Пример:** `area(3) → 28.27`
- **perimeter(r)** — вычисляет длину окружности.  
  **Пример:** `perimeter(3) → 18.85`
